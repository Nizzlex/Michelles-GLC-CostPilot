# Michelle's GLC CostPilot

Vollständig nutzbarer Kostenrechner für den Mercedes GLC 300e (X253).

## Funktionen
- Berechnen-Button
- Preisübersicht für Elektro, kombiniert und Benzin
- Empfehlung der günstigsten Fahrweise
- Jahreskosten, Ersparnis und Break-even
- Speicherung im Browser
- Smartphone- und Desktop-Design

## GitHub
Lade alle vier Dateien in das Stammverzeichnis deines Repositorys.

## GitHub Pages
