# Michelle's GLC CostPilot v1.1

Kostenrechner für den Mercedes GLC 300e (X253).

## Neu in Version 1.1

- Live-Berechnung bei jeder Eingabe
- Berechnen-Button bleibt als klare Bestätigung erhalten
- Dashboard-Kachel „Heute lohnt sich“
- Akkuzustand als eigene Kachel und Eingabewert
- Kosten pro Kilometer und pro 100 km
- Monats- und Jahreskosten
- Rangliste der drei Fahrmodi
- Break-even-Strompreis
- Ersparnis pro 100 km und pro Jahr
- Verbesserte Darstellung auf dem iPhone
- Automatische lokale Speicherung

## Installation des Updates auf GitHub

1. ZIP-Datei entpacken.
2. Im Repository `Michelles-GLC-CostPilot` **Add file → Upload files** öffnen.
3. `index.html`, `styles.css`, `app.js` und `README.md` hochladen.
4. GitHub fragt, ob bestehende Dateien ersetzt werden sollen. Dies bestätigen.
5. Commit-Nachricht, zum Beispiel:
   `Update auf Version 1.1`
6. **Commit changes** auswählen.

GitHub Pages aktualisiert die veröffentlichte Seite normalerweise nach kurzer Zeit automatisch.

## Datenschutz

Alle Werte werden nur im Local Storage des verwendeten Browsers gespeichert. Es findet keine Datenübertragung statt.
